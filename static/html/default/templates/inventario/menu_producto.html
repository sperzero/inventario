Bienvenido al panel de edición de productos

Para registrar productos es necesario crear un documento. Que es un documento?, los documentos permiten enfocarte sólo en la tarea que se está realizando, son temporales por lo que son ideales para verificar los datos antes de ingresarlos permanetemente en el sistema.

Los documentos permiten agrupar tareas grandes, de manera que se pueda hacer en períodos de tiempo separados, es decir, con pausas entre una y otra inserción de nuevos registros.

Para insertar crear un documento seleccione la opción "Nuevo documento" del panel lateral (está marcada con el icono ) y confirme la creación si está de acuerdo

Para insertar productos seleccione la opción "ver documentos" del panel lateral (está marcada con el icono ), escoga un documento para utilizarlo durante la inserción. Si no hay documentos puede cear uno.
